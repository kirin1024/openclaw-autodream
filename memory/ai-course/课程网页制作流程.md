# AI课程网页制作流程

## 概述
- 目的：生成精美的HTML课程页面，部署到局域网供手机/电脑访问
- 存放路径：`~/.openclaw/workspace/ai-course/`
- 访问地址：`http://192.168.1.36:8080`

## 制作流程

### 1. 准备内容
- 先有详细的课程大纲（详细到每个知识点）
- 参考原图或文档确认每个模块的内容

### 2. 编写HTML

**首页 (index.html) 结构**：
```html
<!-- 课程列表，每个项目一个卡片 -->
<a href="project1.html" class="course-card">
  <span class="course-number">1</span>
  <span class="course-title">意图识别技术与实战</span>
  <span class="badge">⭐ 重点</span>
</a>
```

**详情页 (project1.html) 结构**：
```html
<!-- 必须包含的模块 -->
<header>返回链接 + 项目编号</header>
<page-title>项目标题</page-title>
<section>项目背景</section>
<section>核心技术</section>      <!-- 必须详细展开 -->
<section>技术原理</section>    <!-- 流程图/表格 -->
<section>项目全流程实战</section> <!-- 4个步骤+具体任务+代码 -->
<section>学习收获</section>
```

### 3. 样式规范

**颜色变量**：
```css
:root {
  --primary: #4F46E5;        /* 主色-蓝色 */
  --primary-light: #818CF8;  /* 浅蓝 */
  --secondary: #10B981;       /* 绿色-成功 */
  --accent: #F59E0B;         /* 黄色-重点 */
  --bg: #F8FAFC;            /* 背景 */
  --card-bg: #FFFFFF;       /* 卡片背景 */
  --text: #1E293B;          /* 文字 */
  --text-light: #64748B;     /* 次要文字 */
  --yellow-bg: #FFFBEB;     /* 黄色背景块 */
}
```

**响应式**：
- max-width: 800px（手机友好）
- viewport meta 必须加

### 4. 关键元素

**模块标题样式**：
```html
<h2 class="section-title">
  <span class="icon">📋</span>
  标题文字
</h2>
```

**黄色重点框**（核心技术/项目实战）：
```html
<div class="yellow-box">
  <h4>🔬 技术名称</h4>
  <p>详细内容...</p>
</div>
```

**代码块**：
```html
<div class="code-block">
  <div class="code-header">文件名.py</div>
  <pre><code>代码内容</code></pre>
</div>
```

**交付物**：
```html
<div class="deliverable">
  <h5>📦 交付物</h5>
  <p>具体内容</p>
</div>
```

## 部署

### 启动服务
```bash
cd ~/.openclaw/workspace/ai-course
python3 -m http.server 8080
```

### 获取IP地址
```bash
python3 -c "import socket; s=socket.socket(); s.connect(('www.baidu.com', 80)); print(s.getsockname()[0])"
# 输出：192.168.1.36
```

### 访问
- 电脑：http://192.168.1.36:8080
- 手机：确保同WiFi，访问上面地址

---

## 注意事项

1. **内容详细程度**：不能只列提纲，每个点必须有具体解释、代码示例、交付物
2. **模块完整性**：项目背景、核心技术、技术原理、项目实战、学习收获缺一不可
3. **手机适配**：必须加 viewport meta，宽度限制在800px内
4. **代码高亮**：保持代码样式一致性
5. **下一步链接**：每个详情页末尾要有到代码实战页的链接
