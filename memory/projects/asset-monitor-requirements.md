# 个人持仓管理系统 - 需求文档

## 项目概述
自动监测持仓标的的收盘价、计算盈亏，提供盯盘分析和买卖建议。

## 覆盖标的
- A股 + ETF
- 港股
- 美股

---

## 第一阶段：持仓统计

### 功能
1. **盈亏明细** - 每个持仓标的当日赚/亏多少
2. **汇总统计** - 当天总盈亏（金额 + 百分比）
3. **分类查看** - 按 A股/基金/港股/美股 分类

### 持仓视图
- 当前持仓数量
- 成本价
- 收盘价（自动获取）
- 涨跌幅
- 当日盈亏

### 数据管理
| 数据 | 来源 |
|------|------|
| 持仓数量 | 用户手动提供，后续有变化再告知 |
| 成本价 | 用户手动提供，后续有变化再告知 |
| 收盘价 | 每天自动抓取（大金主查 API/网页） |
| 盈亏 | 系统自动算 |

### 工作流
1. 初始：用户告知持仓列表（标的名字、数量、成本价）
2. 存入 `portfolio.json`
3. 每天收盘后，调度大金主去查各市场收盘价
4. 自动计算盈亏，生成报表
5. 有新增/修改持仓时，用户告知

### 输出形式
- 飞书推送每日报表
- 支持按分类查看

---

## 第二阶段：自动盯盘 + 智能建议

### 监控范围
- 开盘前监控
- 盘中实时监控

### 分析功能
- 深度分析：是否建议补仓 / 获利了结 / 止损 / 止盈
- 根据实时价格给出买入/卖出建议

### 信息来源
- 实时价格
- 实时新闻动态（盘前收集）
- 市场层面信息

### 推送时机
- **盘前**：基于昨晚新闻 + 开盘价预期，做一次深度分析
- **盘中**：实时监控，有大幅波动时推送告警

---

## 实现方案

### 方案B：定时任务 + 文件（推荐）

#### 数据文件
- `~/.openclaw/workspace/memory/portfolio.json` - 持仓数据

#### 定时任务
- 每天收盘后（A股 15:00后）：拉取收盘价，生成盈亏报表
- 每天开盘前（9:00）：新闻收集 + 分析建议
- 盘中：实时价格监控（可选）

#### 执行方式
- 调度大金主（invest-advisor）执行
- 结果通过飞书推送给用户
