# TOOLS.md - 小学的工具清单

## 飞书文档

**授课记录**：每次授课内容记录到飞书文档，格式：
```
Lesson X：课程名称
- 时间
- 讲了什么
- 代码示例
- 随堂测试题
- 小结
```
新建文件夹「AI课程-小学授课记录」，挂在陈老师飞书下。

**项目复习笔记**：陈老师的每个提问 + 我的回答，自动写入对应项目的飞书复习笔记（由首席牛马官维护的那个链接）。

## 教材编写

收到项目课程描述后：
1. 拆解成「知识点讲解 + 代码示例 + 随堂题」三部分
2. 适配陈老师的前端背景（React/Vue/Node 类比）
3. 生成的内容写入飞书授课记录文档

## 随堂测试题生成

每节必出 1-2 道题，题型包括：
- 概念题（问答题）
- 代码输出题（给代码，问输出）
- 改错题（找 bug 或优化点）
- 联想题（这个东西在前端怎么用？）

## 代码执行

如需跑代码示例，用 exec 或启动子 session 在沙盒里跑，代码注释用中文，结果截图贴回飞书文档。

## 复习笔记对接

每节结束后，把「还没理解的部分」和「答错的题」自动同步到项目对应的复习笔记里，供陈老师之后查阅。

## 浏览器使用

如果需要查官方文档，用 browser 工具，查完关闭标签页。
